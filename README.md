# 📊 Unemployment Analysis in India

## 🧾 Project Overview
This project focuses on analyzing unemployment trends in India using Python. The goal is to explore how unemployment rates vary across regions, time periods, and during major events like the COVID-19 pandemic. The analysis helps in understanding economic patterns and identifying the most affected areas.

---

## 🎯 Objective
- To study unemployment trends across different states/regions in India  
- To compare unemployment before and during the COVID-19 pandemic  
- To identify regions with high unemployment rates  
- To derive insights for policy and economic improvements  

---

## 📂 Dataset Information
Two datasets are used in this project:

- Unemployment in India.csv  
- Unemployment_Rate_upto_11_2020.csv  

### Key Features:
- Region / State  
- Date  
- Estimated Unemployment Rate (%)  
- Estimated Employed  
- Estimated Labour Participation Rate (%)  
- Area (Rural/Urban)  

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Google Colab / Jupyter Notebook  

---

## 📊 Project Workflow
1. Importing datasets  
2. Data cleaning and handling missing values  
3. Merging datasets  
4. Exploratory Data Analysis (EDA)  
5. Visualization of trends  
6. Extracting insights and conclusions  

---

## 📈 Key Insights
- Unemployment rates varied significantly across different regions  
- States like Haryana, Jharkhand, and Puducherry showed higher unemployment  
- Unemployment increased sharply during the COVID-19 pandemic  
- Rural and urban areas showed different patterns  
- Seasonal variations were observed in employment trends
